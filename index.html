RC Mason 456

Analysis of RCC Structures.

RC456Mason is an app that helps to run on-site analysis of basic RCC structure and its elements using limit state of collapse method (IS 456-2000). Further, users can also use this app for design optimization to conserve both cost and energy intensive materials such as cement and steel beside its compliance as per IS code for structure integrity and safety.
